Jupyter Project

Evolved from the IPython Project

The language-agnostic parts of IPython are getting a new home in Project Jupyter

IPython

  • Interactive Python shell
  • Python kernel for Jupyter
  • Interactive Parallel Python

Jupyter

  • Rich REPL Protocol
  • Notebook (format, environment, conversion)
  • JupyterHub (multi-user notebook server)
  • More...

IPython will continue to exist as a Python kernel for Jupyter, but the notebook and other language-agnostic parts of IPython will move to new projects under the Jupyter name. IPython 3.0 will be the last monolithic release of IPython.
